# UI Features - Dark Mode

## Dark Mode Availability

Dark mode is available for all Pro and Enterprise plans.

## How to Enable Dark Mode

1. Go to Settings > Appearance
2. Select "Dark" from the theme dropdown
3. Changes apply immediately

## System Default Option

If you select "System Default":
- The app will follow your operating system's theme setting
- On macOS: Follows System Preferences > Appearance
- On Windows: Follows Windows Settings > Personalization > Colors
- On Linux: Follows desktop environment theme

## Known Issues

### System Default Not Working
- Some systems may not properly detect dark mode
- Try manually selecting "Dark" instead
- This is a known issue being investigated

### Dark Mode Not Available
- Free plan users do not have access to dark mode
- Upgrade to Pro or Enterprise to enable dark mode
- Dark mode is included in all paid plans

## Feature Requests

### Scheduled Dark Mode
- Currently not available
- Feature request has been logged
- Consider using system-level scheduling tools as workaround
- Check our roadmap for future updates

## Troubleshooting

If dark mode isn't working:
1. Ensure you're on Pro or Enterprise plan
2. Clear app cache
3. Restart the application
4. Contact support if issue persists
